## 2. License Delivery and Activation

1. Customer selects a package and accepts Terms of Sale, EULA, and Privacy Policy
2. Customer pays the yearly license fee
3. Customer installs DitakNet on their own server (Docker / TrueNAS or equivalent)
4. DitakNet generates an Installation ID on the customer server
5. Customer submits the Installation ID to the vendor
6. Vendor issues a signed offline license file or activation code bound to that Installation ID
7. Customer uploads or enters the license in the DitakNet dashboard
8. Backend enforcement unlocks only the purchased modules and limits

Activation codes and license files are non-transferable and bound to one Installation ID unless the vendor approves a documented reinstall or migration.
